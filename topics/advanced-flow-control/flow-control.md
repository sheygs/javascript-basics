## Async & Await

> function declaration

```js
async function fd() {}
```

> arrow function

```js
const arrowFn = async () => {};
```

> callback function

```js
window.addEventListener("click", async function() {});
```

> methods

```js
const person = {
  // method
  sayHi: async function() {},

  // method shorthand
  async sayHello() {},

  // arrow function property
  sayHey: async () => {}
};
```
